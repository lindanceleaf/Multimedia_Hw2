# Multimedia Hw2

## 執行方式
該資料夾中有3個.py檔
- Main.py : 可以一次執行所有的.py
- Q1andQ2.py : 為題目的a, b題所設計的
- Q4.py : 為題目的d題所設計的

若要一次執行，在terminal輸入
```bash
python Main.py
```
即可一次性執行完所有作業的要求

或者要分開執行題目a, b跟d，可以分別輸入
```bash
python Q1andQ2.py
python Q4.py
```
